Як користуватись
Скачуємо *.js файл з Google диску або GitHub, підключаємо до сторінки. На початку кожного скрипта знаходиться функція в яку можна передати необхідні аргументи.
postDate('dateFormat', 'ru', false); // dateFormat (string), language (string), abbreviated (bool). Default: 'dd.mm.yyyy', 'ru', false

Додаємо клас "date-N", де N - кількість "відмотаних" днів.
Наприклад, span class="date-0" - мотає 0 днів назад (сьогодні).

Вивід дати (+ години + хвилини), додаємо клас "time". Наприклад, span class="date-1 time".




Працює як в порядку зростання, так і в порядку спадання (міняємо флажок в функції):




Можна задавати формат часу для окремих дат. Зазвичай, на початку пре-лендінг сторінки йде дата публікації статті, а внизу сторінки - коментарі з датою. Додаємо data-format="format" до тегу з класом date.

Наприклад:
span class="date-8"
span class="date-8"
span class="date-8"
span class="date-8" data-format="day dd, month yyyy"




Передаємо другим аргументом функції назву країни: postDate('dd month yyyy', 'fr');

20 Février 2018

Передаємо третій аргумент функції true, якщо необхідне скорочення до трьох символів: postDate('dd month yyyy', 'de', true);

postSpoiler('spoil', ['Показати фото', 'Сховати фото'], ['spoiler-text', 'spoiler-content']); // Arguments: spoilerName (string), textArray (array), classesArray (array).
// Default: 'spoil', ['Показати фото', 'Сховати фото'], ['spoiler-text', 'spoiler-content']

Блок з текстом - spoil. Всередині нього блок з текстом (spoiler-text) та наступний по DOM-дереву блок с прихованим контентом (spoiler-content)

Блок-контейнер, внутрішні блоки та текст можна змінювати передаючи аргументи при виклику функції.

Наприклад:

Показати фото
Показати фото

Можна викликати функцію для зміну тексту, передаючи другий аргумент в функцію у вигляди масиву рядків: ["Show photo", "Hide photo"] - postSpoiler('', ['Show photo', 'Hide photo']);

Можна задати назви блоків, щоб не копатись в html-коді, якщо спойлер вже є на сторінці: - postSpoiler('', '', ['text', 'conent'];

postCalendar(calendarId);

Календар задається у формі таблиці id="calendar" (можна змінити в разі необхідності).

Копіюємо html-розмітку та стилі. Отримуємо:

postFixedSidebar(className) // Argument: className (string). Default: 'sticky'

Задаємо клас sticky для блоку, який хочемо зафіксувати + копіюємо стилі для коректного відображення.

При потребі, клас можна задати свій. Тоді змінюємо css-стилі для свого класу .className-wrapper та .className-wrapper.fixed. Частину wrapper не потрібно видаляти. Вона автоматично додається для коректного відображення.

Приклад:

postPopup();

Копіюємо html-розмітку в кінець сторінки та стилі (або підганяємо під дизайн сторінки самостійно). Тикаємо по кнопці і виводимо мишку за межі екрану вверх. В результаті отримуємо:
